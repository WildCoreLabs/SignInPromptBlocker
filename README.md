# SignInPromptBlocker (iOS Safari Web Extension)

Blocks Google One Tap / “Sign in with Google” prompts in Safari.

## Features
- Removes Google Identity Service iframes & containers
- Per-site allowlist
- **Smart Redirects (optional):** Open Google Maps links in Apple Maps

## How to enable the extension
On iPhone/iPad: Settings → Safari → Extensions → SignInPromptBlocker → ON → Allow on All Websites.

## Smart Redirects
Open the extension options and toggle “Open Google Maps links in Apple Maps”.
